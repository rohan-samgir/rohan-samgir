## 🚀 Featured Projects

### 🔧 [vProfile — AWS Cloud Re-Architecture (PaaS/SaaS)](https://github.com/rohan-samgir)
> Migrated a monolithic Java app to a fully managed AWS stack with auto-scaling and zero-downtime deployments.

```
Stack: Elastic Beanstalk · RDS · ElastiCache · Amazon MQ
       CloudFront · ECS · EKS · Docker · Terraform · ACM
```
- ✅ Containerized with Docker, orchestrated on EKS/ECS using `kubectl`
- ✅ Decoupled backend using Amazon RDS, ElastiCache & Amazon MQ
- ✅ CloudFront CDN + SSL/TLS via AWS Certificate Manager
- ✅ Full IaC provisioning with Terraform

```
Stack: Nginx · Apache Tomcat · RabbitMQ · Memcached · MySQL
       CentOS · VirtualBox · Bash · Firewalld
```

- ✅ End-to-end stack performance validated under simulated load
